## Pre-requisites:
* install Python Extension and follow tutorial: https://code.visualstudio.com/docs/python/python-tutorial
* To install MkDocs run following command:
```
pip install mkdocs
pip install mkdocs-material
```
* 

## Running locally
* run following:
```
mkdocs serve
```
